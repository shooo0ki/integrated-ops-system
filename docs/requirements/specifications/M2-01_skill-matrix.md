# M2-01 スキルマトリクス

## 目的
全メンバーのスキル・資質レベルをマトリクス形式で一覧表示し、アサイン候補の絞り込みを支援する。

> **スキルカテゴリ方針（変更）**:
> 細かいスキル項目より大まかな分類を優先。技術スキルに加え、**定性的な評価項目**も含める。
>
> **推奨カテゴリ例:**
> - フロントエンド開発
> - バックエンド開発
> - ビジネス / コンサルティング
> - **定性評価**（熱意・コミットメント量・アピアランス・コミュニケーション 等）

## 対象ペルソナ / シナリオ
- 田中（マネージャー）・佐藤（管理者）: スキル確認 → PJ アサイン

## UI 構成
- フィルターバー（スキルカテゴリ / 最低レベル）
- マトリクステーブル
  - 行: メンバー名
  - 列: スキル項目（カテゴリでグループ化）
  - セル: レベル値（1〜5）を色分け表示（未評価はグレー）
- 評価入力ボタン（管理者・マネージャーのみ）
- スキル設定リンク（管理者のみ）

## スキルレベル定義
| レベル | 定量評価例 | 定性評価例（熱意の場合） |
|--------|-----------|----------------------|
| 1 | 初学者・未経験 | 関心薄・受け身 |
| 2 | 基礎知識あり | 言われればやる |
| 3 | 実務で自走可能 | 自発的に取り組む |
| 4 | 熟練・指導可能 | 周囲を巻き込む |
| 5 | エキスパート | チームの推進力 |

## 表示項目
| 項目 | 説明 |
|------|------|
| メンバー名 | クリックで M1-02 へ遷移 |
| スキルレベル（1〜5） | 色: 1=グレー / 2=黄 / 3=緑 / 4=青 / 5=紫 |
| 未評価 | 「—」表示（グレー背景） |

## 操作
| 操作 | 説明 | 遷移先 |
|------|------|--------|
| カテゴリフィルター | 特定カテゴリの列のみ表示 | — |
| レベルフィルター | 条件を満たすメンバーのみ行表示 | — |
| 評価入力アイコン | 該当メンバーのスキル評価入力を開く | M2-03 |
| メンバー名クリック | メンバー詳細を開く | M1-02 |
| スキル設定ボタン | スキルカテゴリ・項目の管理 | M2-02 |

## 権限（ロール別）
| ロール | 閲覧 | 評価編集 |
|--------|------|---------|
| 管理者 | 全員 | 全員 |
| マネージャー | 全員 | 担当 PJ メンバーのみ |
| メンバー（社員・インターン共通） | 全員（閲覧のみ） | 不可 |

## 関連データ（エンティティ）
- `MEMBER`、`SKILL`、`SKILL_CATEGORY`、`MEMBER_SKILL`

## 関連 API
- `GET /api/skills/matrix?category=&minLevel=`
