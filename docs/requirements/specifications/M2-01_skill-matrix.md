# M2-01 スキルマトリクス

## 目的
全メンバーのスキルレベルをマトリクス形式で一覧表示し、アサイン候補の絞り込みを支援する。失敗シナリオ F3（スキル陳腐化）を防ぐため、最終更新日も可視化する。

## 対象ペルソナ / シナリオ
- 田中（マネージャー）・佐藤（管理者）: S3 スキル確認 → PJ アサイン

## UI 構成
- フィルターバー（スキルカテゴリ / 最低レベル / 所属会社）
- マトリクステーブル
  - 行: メンバー名
  - 列: スキル項目（カテゴリでグループ化）
  - セル: レベル値（1〜5）を色分け表示（未評価はグレー）
- 最終更新日列（3 ヶ月以上未更新はオレンジ強調）
- スキル評価入力ボタン

## 表示項目
| 項目 | 説明 |
|------|------|
| メンバー名 | クリックで M1-02 へ遷移 |
| スキルレベル（1〜5） | 色: 1=グレー / 2=黄 / 3=緑 / 4=青 / 5=紫 |
| 未評価 | 「—」表示（グレー背景） |
| 最終スキル更新日 | 3 ヶ月超過でオレンジ表示 |

## 操作
| 操作 | 説明 | 遷移先 |
|------|------|--------|
| カテゴリフィルター | 特定カテゴリの列のみ表示 | — |
| レベルフィルター（最低 X 以上） | 条件を満たすメンバーのみ行表示 | — |
| セルクリック | 該当メンバーのスキル評価入力を開く | M2-03 |
| メンバー名クリック | メンバー詳細を開く | M1-02 |

## エラー / 例外
| 条件 | 挙動 |
|------|------|
| スキル項目が 0 件 | 「スキルが未登録です。M2-02 でスキルを追加してください」 |

## 権限（ロール別）
| ロール | 閲覧 | 評価編集 |
|--------|------|---------|
| 管理者 | 全員 | 全員 |
| マネージャー | 全員 | 担当 PJ メンバーのみ |
| 社員 | 全員（閲覧のみ） | 不可 |
| インターン | 自分のみ | 不可 |

## 関連データ（エンティティ）
- `MEMBER`、`SKILL`、`SKILL_CATEGORY`、`MEMBER_SKILL`、`SKILL_LEVEL`

## 関連 API
- `GET /api/skills/matrix?category=&minLevel=&company=`
