# M6-02 全社 PL サマリー

## 目的
ブーストコンサルティング / SALT2 の全 PJ を合算した会社全体の損益を確認する。経営判断の起点。

## 対象ペルソナ / シナリオ
- 佐藤（管理者）: 月初の全社 PL 確認・経営判断

## UI 構成
- 会社フィルター（ブーストコンサルティング / SALT2 / 両社）
- 対象月選択
- 全社 PL テーブル（PJ 別の明細 + 合計行）
- 月次推移グラフ（折れ線: 売上・コスト・粗利）
- PJ 別粗利率ランキング

## 表示項目
| 項目 | 説明 |
|------|------|
| 会社名 | — |
| PJ 名 | — |
| 売上合計 | — |
| コスト合計 | — |
| 粗利 | — |
| 粗利率 | % |
| 全社合計行 | 全 PJ の合計 |

## 操作
| 操作 | 説明 |
|------|------|
| 会社フィルター | 表示会社を切り替え |
| 月切り替え | 前月・翌月ナビ |
| PJ 名クリック | M6-01 PJ 別 PL へ遷移 |

## エラー / 例外
| 条件 | 挙動 |
|------|------|
| PJ が 0 件 | 「進行中のプロジェクトがありません」 |

## 権限（ロール別）
| ロール | アクセス |
|--------|---------|
| 管理者 | フルアクセス |
| その他 | アクセス不可 |

## 関連データ（エンティティ）
- `PL_RECORD`、`PROJECT`

## 関連 API
- `GET /api/pl/summary?month=&company=`
