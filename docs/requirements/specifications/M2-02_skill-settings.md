# M2-02 スキル設定

## 目的
スキルカテゴリとスキル項目のマスタデータを管理者が追加・編集・並べ替えできる。

## 対象ペルソナ / シナリオ
- 佐藤（管理者）: 事業拡大に伴うスキル項目の追加・整理

## UI 構成
- スキルカテゴリ一覧（ドラッグ&ドロップで並べ替え）
- カテゴリ内スキル項目一覧（ドラッグ&ドロップで並べ替え）
- カテゴリ追加フォーム
- スキル項目追加フォーム

## 入力項目
| 項目 | 必須 | 制約 | 例 |
|------|------|------|----|
| カテゴリ名 | 必須 | 最大 50 文字・一意 | `エンジニアリング` |
| カテゴリ説明 | 任意 | 最大 200 文字 | — |
| スキル名 | 必須 | 最大 100 文字・カテゴリ内一意 | `フロントエンド` |
| スキル説明 | 任意 | 最大 200 文字 | `React, Vue.js 等` |
| 所属カテゴリ | 必須 | 既存カテゴリから選択 | — |

## 操作
| 操作 | 説明 |
|------|------|
| カテゴリ追加 | 新規カテゴリを作成 |
| スキル追加 | カテゴリに新規スキル項目を追加 |
| 編集 | カテゴリ名・スキル名の変更 |
| 削除 | 評価データが存在しない場合のみ削除可 |
| 並べ替え | ドラッグ&ドロップで表示順を変更 |

## エラー / 例外
| 条件 | 挙動 |
|------|------|
| カテゴリ名重複 | 「同名のカテゴリがすでに存在します」 |
| 評価データありのスキル削除 | 「このスキルには評価データが存在するため削除できません」 |

## 権限（ロール別）
| ロール | アクセス |
|--------|---------|
| 管理者 | フルアクセス |
| その他 | アクセス不可 |

## 関連データ（エンティティ）
- `SKILL_CATEGORY`、`SKILL`、`SKILL_LEVEL`

## 関連 API
- `GET /api/skill-categories`
- `POST /api/skill-categories`
- `PUT /api/skill-categories/:id`
- `DELETE /api/skill-categories/:id`
- `POST /api/skills`
- `PUT /api/skills/:id`
- `DELETE /api/skills/:id`
