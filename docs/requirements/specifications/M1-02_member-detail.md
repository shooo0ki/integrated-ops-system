# M1-02 メンバー詳細

## 目的
特定メンバーの詳細情報（プロフィール・報酬・権限）を閲覧・管理する。

## 対象ペルソナ / シナリオ
- 佐藤（管理者）: 報酬設定・権限変更
- 鈴木・山田: 自分の情報確認・編集

## UI 構成
- プロフィールヘッダー（アバター・氏名・ステータスバッジ）
- 基本情報セクション
- 報酬情報セクション（管理者のみ表示）
- 銀行口座情報セクション（管理者 + 本人のみ）
- アサイン中プロジェクト一覧
- 利用ツール一覧（ツール名 / プラン / 月額 / 請求先会社）
- 契約書セクション（契約書リスト・ステータス・署名済PDFダウンロード）
- スキル評価サマリー
- 編集ボタン

## 表示項目
| 項目 | 管理者 | マネージャー（担当 PJ メンバー） | 社員・インターン（本人のみ） |
|------|--------|--------------------------------|---------------------------|
| 氏名・メール・電話 | ○ | ○ | ○ |
| 住所 | ○ | — | ○ |
| 入社日・退社日 | ○ | ○ | ○ |
| ステータス・ロール | ○ | ○ | ○（閲覧のみ） |
| 報酬形態・報酬金額 | ○ | — | — |
| 銀行口座情報 | ○ | — | ○（本人のみ） |
| アサイン PJ 一覧 | ○ | ○（担当 PJ のみ） | ○（本人のみ） |
| 利用ツール一覧 | ○ | ○（担当 PJ のみ） | ○（本人のみ） |
| 契約書（署名済PDF/ステータス） | ○ | — | ○（本人のみ） |
| 口座情報 | ○ | — | ○（本人のみ） |
| スキル評価 | ○ | ○ | ○（本人のみ） |

## 操作
| 操作 | 説明 | 遷移先 |
|------|------|--------|
| 編集ボタン | 編集フォームを開く | M1-03 |
| PJ 名クリック | PJ 詳細を開く | M3-02 |
| スキル評価クリック | スキル評価入力を開く | M2-03 |

## エラー / 例外
| 条件 | 挙動 |
|------|------|
| 存在しない ID | 404 ページ |
| 権限外のメンバーにアクセス | 403 ページ |

## 権限（ロール別）
- 管理者: 全員・全項目
- マネージャー: 担当 PJ メンバーのみ・限定項目
- 社員・インターン: 自分のみ

## 関連データ（エンティティ）
- `MEMBER`、`USER_ACCOUNT`、`PROJECT_ASSIGNMENT`、`MEMBER_SKILL`

## 関連 API
- `GET /api/members/:id`
- `GET /api/members/:id/assignments`
- `GET /api/members/:id/skills`
