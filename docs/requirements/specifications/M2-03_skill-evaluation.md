# M2-03 スキル評価入力

## 目的
特定メンバーのスキルレベルを設定・更新する。評価根拠メモと評価日を記録して変遷を追跡できる。

## 対象ペルソナ / シナリオ
- 田中（マネージャー）・佐藤（管理者）: S3 アサイン前のスキル確認・更新

## UI 構成
- 対象メンバー情報ヘッダー
- スキルカテゴリタブ
- スキル項目ごとの評価入力行（レベル選択 + メモ + 評価日）
- スキル評価履歴リスト（アコーディオン）
- 保存ボタン

## 入力項目
| 項目 | 必須 | 制約 | 例 |
|------|------|------|----|
| スキルレベル | 必須 | 1〜5 の整数 | `3` |
| 評価日 | 必須 | 過去日 or 当日 | `2026-02-18` |
| 評価メモ | 任意 | 最大 500 文字 | `React 実務 1 年。中規模開発で自走可能` |

## 表示項目
| 項目 | 説明 |
|------|------|
| 現在のレベル | 最新の評価値 |
| レベルラベル | 初学者 / 初級 / 中級 / 上級 / エキスパート |
| 評価履歴 | 評価日・レベル・メモの時系列リスト |

## 操作
| 操作 | 説明 |
|------|------|
| レベル選択 | 1〜5 ボタン or セレクトボックス |
| 保存 | 評価を登録（履歴として追記） |
| 履歴展開 | 過去の評価履歴を展開表示 |

## エラー / 例外
| 条件 | 挙動 |
|------|------|
| レベル未選択で保存 | 「レベルを選択してください」 |

## 権限（ロール別）
| ロール | 操作 |
|--------|------|
| 管理者 | 全メンバー評価可 |
| マネージャー | 担当 PJ メンバーのみ |
| 社員・インターン | 閲覧のみ（自分のみ） |

## 関連データ（エンティティ）
- `MEMBER_SKILL`、`SKILL`、`MEMBER`

## 関連 API
- `GET /api/members/:id/skills`
- `PUT /api/members/:id/skills/:skillId`
