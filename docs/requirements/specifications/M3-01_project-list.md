# M3-01 プロジェクト一覧

## 目的
全プロジェクトを一覧表示し、ステータス・会社でフィルタリングして目的の PJ をすばやく見つける。

## 対象ペルソナ / シナリオ
- 佐藤（管理者）・田中（マネージャー）: PJ 状況の全体把握

## UI 構成
- フィルターバー（ステータス / 所属会社 / 検索ワード）
- PJ カード or テーブル一覧
- 新規 PJ 作成ボタン（管理者・マネージャーのみ）

## 表示項目（一覧行）
| 項目 | 説明 |
|------|------|
| プロジェクト名 | — |
| ステータスバッジ | 計画中 / 進行中 / 完了 / 保留 |
| 所属会社 | ブーストコンサルティング / SALT2 |
| クライアント名 | — |
| 開始日・終了予定日 | — |
| アサイン人数 | 現在の稼働メンバー数 |

## 操作
| 操作 | 遷移先 |
|------|--------|
| PJ 行クリック | M3-02 プロジェクト詳細 |
| 新規作成ボタン | M3-03 PJ 登録 |
| フィルター適用 | 即時絞り込み |

## エラー / 例外
| 条件 | 挙動 |
|------|------|
| 検索結果 0 件 | 「該当するプロジェクトが見つかりません」 |

## 権限（ロール別）
| ロール | 閲覧範囲 |
|--------|---------|
| 管理者 | 全 PJ |
| マネージャー | 全 PJ（閲覧） |
| 社員・インターン | アサインされている PJ のみ |

## 関連データ（エンティティ）
- `PROJECT`、`PROJECT_ASSIGNMENT`

## 関連 API
- `GET /api/projects?status=&company=&q=`
