# M7-01 人事評価一覧

## 目的
PAS（Professional / Appearance / Skill）3軸による月次人事評価を管理・閲覧する。
管理者は全メンバーの評価を参照・編集できる。社員・インターンは自分の評価のみ閲覧できる。

## 対象ペルソナ / シナリオ
- 佐藤（管理者）: 月次評価を入力し、メンバー全員の成長状況を把握する
- 鈴木（社員）: 自分の最新評価を確認する

## UI 構成

### 管理者ビュー
- 対象月セレクト（直近12ヶ月）
- メンバーテーブル（行: メンバー / 列: P・A・S スコア・コメント・評価日・操作）
- [評価入力] ボタン → M7-02 評価入力モーダル

### 社員・インターンビュー
- 自分の評価履歴のみ表示（月別カード形式）
- 各月のP/A/S スコア + レーダーチャート
- 管理者コメント

## 表示項目

| 項目 | 管理者 | 社員・インターン |
|------|--------|-----------------|
| メンバー名 | 全員 | 自分のみ |
| 対象月 | ○ | ○ |
| P スコア（1〜5） | ○ | ○ |
| A スコア（1〜5） | ○ | ○ |
| S スコア（1〜5） | ○ | ○ |
| 総合スコア（P+A+S 平均） | ○ | ○ |
| 評価コメント | ○ | ○ |
| 評価者名 | ○ | ○ |
| 評価日 | ○ | ○ |
| 編集ボタン | ○（全員） | ✕ |

## スコア定義

| スコア | ラベル | 色 |
|--------|--------|----|
| 1 | 要改善 | 赤 |
| 2 | 普通以下 | オレンジ |
| 3 | 標準 | 黄 |
| 4 | 優秀 | 緑 |
| 5 | 卓越 | 青 |

## 評価軸（PAS）

| 軸 | 名称 | 評価内容 |
|----|------|---------|
| P | Professional（プロフェッショナル） | 責任感・期限厳守・コミュニケーション・チームへの貢献 |
| A | Appearance（アピアランス） | 清潔感・身だしなみ・社会人としての立ち居振る舞い |
| S | Skill（スキル） | 専門技術力・業務遂行能力の総合評価 |

## 操作
| 操作 | 説明 | 権限 |
|------|------|------|
| 対象月切替 | セレクトで月を変更 | 全員 |
| 評価入力 | 新規 or 上書き評価 → M7-02 モーダル | 管理者のみ |
| 評価編集 | 既存評価を修正 → M7-02 モーダル | 管理者のみ |
| 履歴閲覧 | 過去月の評価を閲覧 | 全員（管理者:全員分 / その他:自分のみ） |

## エラー / 例外
| 条件 | 挙動 |
|------|------|
| 未評価月 | 「未評価」と表示（—グレー） |
| 権限外アクセス | 403 返却、画面リダイレクト |

## 権限（ロール別）
| ロール | 閲覧範囲 | 編集 |
|--------|----------|------|
| 管理者 | 全メンバー | ○ |
| マネージャー | 全メンバー（閲覧のみ） | ✕ |
| 社員・インターン | 自分のみ | ✕ |

## 関連データ（エンティティ）
- `PERSONNEL_EVALUATION`
- `MEMBER`
- `USER_ACCOUNT`（評価者情報）

## 関連 API
- `GET /api/evaluations?month=YYYY-MM` — 管理者: 全員, その他: 自分のみ
- `GET /api/evaluations/:memberId` — メンバー別評価履歴
